### # Install Genymotion

```
$ sudo apt-get update

$ sudo apt-get install virtualbox

# download genymotion
$ ls -l ~/Downloads/ | grep geny

$ chmod +x  genymotion...xxx.bin

$ sudo ~/Downloads/genymotion...xxx.bin
```
